## 🚀 QUICK ACTION ITEMS

### ✅ Avant de Deployer
```bash
# 1. Vérifier que ça marche en local
npm run dev
# Teste sur http://localhost:3000

# 2. Vérifier que la build fonctionne
npm run build
npm start

# 3. Vérifier Git
git status  # Pas de .env.local!
```

### ✅ Pour Deployer
```bash
# 1. Push sur GitHub
git add .
git commit -m "Initial commit"
git branch -M main
git push origin main

# 2. Importe dans Vercel
# https://vercel.com/new

# 3. Configure variables
# Vercel → Settings → Environment Variables

# 4. Attend la build
# Vercel → Deployments → Ready ✅

# 5. Partage le lien!
```
